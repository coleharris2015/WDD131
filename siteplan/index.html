Overview

Purpose

The purpose of this website is to make an enjoyable game with information on Transformers.

Audience

The intended audience is anyone that wants to have fun or learn about Transformer

Dynamic elements

I will use javascript to make a simple game loop and to load up the transformer of a players choice out of an array.

Upon selection whether the Transformer is an Autobot or a Decipticon will determine the enemies that will be fought

The character cards will have two images each one for robot and alt mode and the player can change their robots mode.

Damage will depend on stats the slected character has and will be determined with javascript math functions

Before selecting a robot the player will be able to browse a menu with bios of the robots that shows stats and what they look like
